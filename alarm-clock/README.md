# Alarm Clock

This module contains the code for:
1. `alarm.py`: The file which sets alarms on the system using Tkinter and time module
2. `clock.puy`: A simple program which displays the current digital time in a widget

Both these files use Tkinter and native Python Library to achieve the task.
The coding style used is Google's Style of Code and resembles a production
codebase that is made easy for reading.
