# Youtube Downloader

This module contains the YouTube Video Downloader CLI app. It fetches data from
YouTube and
stores the files to hard disk in various foramts

It uses `argparse` to crete and build the cli and then uses python I/O to store
the file.

Also written are the test for the program in the `test_youtube_downloader.py`
file. It uses
`pytest` as a test runner and `unittest` for mocking and assertions
